## 🚫 Критические ошибки (избегать любой ценой)

### Контент и информация

| ❌ Ошибка | ✅ Решение |
|-----------|------------|
| Мало информации — непонятно что это | Главный заголовок объясняет всё за 5 секунд |
| Жаргон и термины компании | Говорим языком пользователей: "койко-место", "от 600₽" |
| Негативное начало ("Не платите...") | Позитивный посыл: "Уютный хостел от 600₽" |
| Цены/оплата слишком рано | Сначала покажи ценность, потом цену |
| Ссылки на внешние сайты вместо своих страниц | Вся информация на нашем сайте |

### Дизайн и UI

| ❌ Ошибка | ✅ Решение |
|-----------|------------|
| Пурпурные градиенты (AI slop) | Уникальная палитра с характером |
| Шаблонный дизайн | Кастомные решения под контекст |
| Разные стили на разных страницах | Единая дизайн-система |
| Маленький/невнятный логотип | Читаемый логотип, можно wordmark |
| Много кнопок и CTA на экране | Один главный CTA на каждый viewport |
| Движущиеся элементы при чтении | Анимации только для акцентов, не отвлекают |
| Hijack scroll | Нативный скролл, никаких перехватов |
| Разные размеры/стили скриншотов | Консистентность во всех изображениях |
| Линии там где не нужны | Разделение через пространство, не линии |

### UX и взаимодействие

| ❌ Ошибка | ✅ Решение |
|-----------|------------|
| Требование регистрации сразу | Позволяем проверить даты без регистрации |
| Пустые состояния с бесконечными спиннерами | Показываем прогресс, что происходит |
| Скрытая информация (hover-only) | Важное видно сразу |
| Много шагов до цели | Минимум кликов до бронирования |

## ✅ Принципы хорошего UX (что делаем)

### 1. Ясность выше всего
```
Headline должен быть понятен БЕЗ subheadline.
Если нужен subheadline для понимания — переписать headline.
```

### 2. Один CTA на viewport
```
Каждый экран (при скролле) имеет ОДНУ главную кнопку.
Приоритет: Главное сообщение → CTA → Доказательства
```

### 3. Язык пользователя
```
❌ "Оптимальное размещение в премиальных условиях"
✅ "Койко-место от 600₽ в центре Сочи"
```

### 4. Покажи, не рассказывай
```
- Фото номеров крупно
- Калькулятор цены работает сразу
- Можно проверить даты без регистрации
```

### 5. Детали решают
```
- Все изображения одного стиля
- Отступы консистентны
- Тени/границы одинаковые
- Шрифты не скачут
```

## 🎨 Цветовая палитра

### Концепция: "Тёплый минимализм у моря"

Отходим от холодных синих и избитых пурпурных. Создаём ощущение **тепла**, **уюта** и **близости к морю** через песочные и терракотовые оттенки.

### Основные цвета

| Название | HEX | RGB | Использование |
|----------|-----|-----|---------------|
| **Песок** | `#F5F0E8` | 245, 240, 232 | Основной фон |
| **Тёплый белый** | `#FDFCFA` | 253, 252, 250 | Карточки, блоки |
| **Терракота** | `#C4704A` | 196, 112, 74 | Главный акцент, CTA |
| **Терракота тёмная** | `#A85A38` | 168, 90, 56 | Hover состояния |
| **Уголь** | `#2D2A26` | 45, 42, 38 | Заголовки, текст |
| **Серый тёплый** | `#6B6560` | 107, 101, 96 | Вторичный текст |
| **Серый светлый** | `#B8B2AA` | 184, 178, 170 | Границы, иконки |

### Акцентные цвета

| Название | HEX | Использование |
|----------|-----|---------------|
| **Морской** | `#3D8B8B` | Ссылки, badges |
| **Успех** | `#4A8C5C` | Подтверждения |
| **Предупреждение** | `#D4A04A` | Важные заметки |
| **Ошибка** | `#C45C4A` | Ошибки валидации |

## 🔤 Типографика

### Концепция: Читаемость + Характер

Избегаем generic шрифтов (Inter, Roboto). Выбираем шрифты с характером, но хорошей читаемостью.

### Шрифты

| Роль | Шрифт | Fallback | Почему |
|------|-------|----------|--------|
| **Заголовки** | **Unbounded** | system-ui | Современный, запоминающийся, хорошая кириллица |
| **Текст** | **Golos Text** | system-ui | Нейтральный, отличная читаемость, русский |
| **Альтернатива заголовков** | Manrope | system-ui | Если Unbounded слишком bold |

### Размеры (Desktop)

| Элемент | Размер | Вес | Line-height |
|---------|--------|-----|-------------|
| H1 (Hero) | 48px / 3rem | 700 | 1.1 |
| H2 (Секции) | 32px / 2rem | 600 | 1.2 |
| H3 (Карточки) | 24px / 1.5rem | 600 | 1.3 |
| Body Large | 18px / 1.125rem | 400 | 1.6 |
| Body | 16px / 1rem | 400 | 1.6 |
| Small | 14px / 0.875rem | 400 | 1.5 |
| Caption | 12px / 0.75rem | 500 | 1.4 |

### Размеры (Mobile)

| Элемент | Размер |
|---------|--------|
| H1 | 32px |
| H2 | 24px |
| H3 | 20px |
| Body | 16px |

### CSS

```css
/* Подключение шрифтов */
@import url('https://fonts.googleapis.com/css2?family=Unbounded:wght@400;500;600;700&display=swap');
@import url('https://fonts.cdnfonts.com/css/golos-text');

:root {
  --font-heading: 'Unbounded', system-ui, sans-serif;
  --font-body: 'Golos Text', system-ui, sans-serif;
}

h1, h2, h3, h4 {
  font-family: var(--font-heading);
}

body {
  font-family: var(--font-body);
}
```

## 📱 Адаптивность

### Breakpoints

| Название | Размер | Устройства |
|----------|--------|------------|
| Mobile | < 640px | Телефоны |
| Tablet | 640px - 1024px | Планшеты |
| Desktop | > 1024px | Десктопы |

### Mobile First

- **80%+ трафика** в туристическом сегменте — мобильные
- Дизайн сначала для мобильных, потом расширяем
- Кнопка звонка (click-to-call) на видном месте
- Форма бронирования адаптирована под тач

## ⚡ Анимации

### Принципы

1. **Анимации для акцентов, не для отвлечения**
2. **Никаких движений при чтении текста**
3. **Быстрые и subtle** (200-300ms)
4. **Prefer reduced motion** — уважаем настройки системы

### Где использовать

| Элемент | Анимация | Длительность |
|---------|----------|--------------|
| Кнопки (hover) | scale(1.02), darken bg | 150ms |
| Карточки (hover) | translateY(-4px), shadow | 200ms |
| Модалки | fade + scale | 250ms |
| Страницы | fade-in | 300ms |
| Успешное бронирование | confetti/checkmark | 500ms |

### CSS

```css
/* Базовый transition */
.btn {
  transition: transform 150ms ease, background-color 150ms ease;
}

.btn:hover {
  transform: scale(1.02);
}

/* Уважаем prefers-reduced-motion */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 🔑 Главные takeaways

1. **Ясность > Красота** — лучше понятно, чем модно
2. **Детали решают** — выверять каждый пиксель
3. **Один CTA** — не перегружать выбором
4. **Mobile first** — 80% пользователей на телефонах
5. **Дай попробовать** — не требуй регистрацию сразу
6. **Язык пользователя** — "койко-место", не "единица размещения"
7. **Покажи продукт** — фото номеров крупно и качественно
